# 🧠 Treino de Lógica de Programação

Repositório criado para armazenar exercícios, desafios e soluções desenvolvidos durante meus estudos de **Lógica de Programação**.

O objetivo é fortalecer o raciocínio lógico e a capacidade de resolver problemas, criando uma base sólida para o desenvolvimento de software.

---

## 🎯 Objetivos

* Desenvolver o raciocínio lógico.
* Praticar estruturas de programação.
* Resolver desafios de diferentes níveis de dificuldade.
* Aprimorar a criação de algoritmos.
* Evoluir constantemente na resolução de problemas.

---

## 📚 Conteúdo

Neste repositório você encontrará exercícios envolvendo:

* ✅ Variáveis
* ✅ Operadores
* ✅ Entrada e saída de dados
* ✅ Estruturas condicionais (`if`, `else`, `switch`)
* ✅ Estruturas de repetição (`for`, `while`, `do while`)
* ✅ Vetores e Matrizes
* ✅ Funções
* ✅ Algoritmos
* ✅ Desafios de lógica
* ✅ Problemas de programação

---

## 📂 Organização

Os exercícios deste repositório estarão organizados por linguagem de programação e tecnologia, tornando a navegação mais simples e facilitando o acompanhamento dos estudos.

---

## 🚀 Objetivo Final

Construir uma base sólida em lógica de programação para facilitar o aprendizado de estruturas de dados, algoritmos e desenvolvimento de software.

---

## 📈 Evolução

Este repositório será atualizado continuamente conforme novos exercícios e desafios forem sendo resolvidos.
